## About MyCenter

MyCenter 意为《个人中心》 或 《个人主页》，为个人而打造的技能服务平台，MyCenter的目标是帮助个人通过互联网推广、运营自身的价值，让自己的技能服务于他人，提升个人的收益、项目经验等。


## 安装方法

- 服务器系统(推荐)：Ubuntu
- PHP 7.4.10+
- Mysql 5.7+
- Redis
- Composer
